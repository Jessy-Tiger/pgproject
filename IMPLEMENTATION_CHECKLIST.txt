## ✅ TROUBLESHOOTING (If Steps Failed)

### Connection Test Failed?

1. [ ] Check internet connection: `ping google.com`
2. [ ] Verify .env file has EMAIL_HOST_PASSWORD
3. [ ] Verify EMAIL_DEBUG_MODE=False (to actually connect)
4. [ ] Run: `python manage.py test_email --verbose`
5. [ ] Read: `EMAIL_TROUBLESHOOTING.md`

### Email Not Received?

1. [ ] Check spam folder in Gmail
2. [ ] Verify correct recipient email in settings
3. [ ] Wait 2 minutes - sometimes slow
4. [ ] Check logs for errors
5. [ ] Run with --verbose flag for details

### Authentication Error?

1. [ ] Generating App Password again?
   - Go to: https://myaccount.google.com/apppasswords
   - Generate NEW password
   - Update .env
   - Restart Django

2. [ ] Using wrong email/password?
   - Verify EMAIL_HOST_USER matches your Gmail
   - Verify EMAIL_HOST_PASSWORD is 16 characters

### Still Not Working?

- [ ] Read: `EMAIL_CONFIGURATION_GUIDE.md` (detailed troubleshooting)
- [ ] Read: `EMAIL_TROUBLESHOOTING.md` (common errors)
- [ ] Search error message in troubleshooting docs
- [ ] Follow step-by-step solution
